# 设计文档 v3.2，第三章第 1 层

# =============================================================================

## 表格名称
活动内容生产工作台

## 字段定义

### 运营填写字段（可见）

| 字段名 | 类型 | 必填 | 选项/格式 | 说明 |
|--------|------|------|-----------|------|
| 活动名称 | 单行文本 | ✅ | — | 海报标题、推文标题、视频标题卡 |
| 活动时间 | 日期时间 | ✅ | YYYY-MM-DD HH:mm | 海报日期、推文正文、视频字幕 |
| 一句话简介 | 单行文本 | ✅ | — | 海报副标题、推文摘要、视频简介卡 |
| 活动详情 | 多行文本 | ✅ | — | 推文正文素材、视频旁白文案来源 |
| 视觉风格 | 单选 | ✅ | 科技蓝 / 开源绿 / 社区红 | AI 生成底图的风格方向 |
| 海报模板 | 单选 | ✅ | 活动预告 / 活动回顾 / 嘉宾海报 | 稿定品牌模板选择 |
| 推文类型 | 单选 | ✅ | 活动招募 / 活动回顾 / 嘉宾访谈 | Dify prompt 模板选择 |
| 生成视频 | 单选 | ❌ | 是 / 否 | 默认否 |
| 视频类型 | 单选 | 条件 | 活动预热短视频 / 嘉宾金句切片 / 科普动画 | 生成视频=是 时必填 |
| 视频时长 | 单选 | 条件 | 15秒 / 30秒 / 60秒 | 生成视频=是 时必填 |
| 视频旁白风格 | 单选 | 条件 | 专业沉稳 / 活力轻快 / 科技未来感 | TTS+BGM，生成视频=是 时必填 |
| 涉及地图 | 单选 | ❌ | 是 / 否 | 是 → 触发地图强制人审 |
| 特殊要求 | 多行文本 | ❌ | — | 透传给审核人 |
| 再生成意见 | 多行文本 | ❌ | — | 审核人修改意见（再生成回环用） |

### 系统字段（n8n 回写，只读）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 海报状态 | 单选 | 生成中 / 待审 / 失败 / 超时失败 / 系统错误 / 校验失败 / 已定稿 |
| 推文状态 | 单选 | 生成中 / 待审 / 失败 / 超时失败 / 系统错误 / 校验失败 / 已定稿 |
| 视频状态 | 单选 | 生成中 / 待审 / 失败 / 超时失败 / 系统错误 / 校验失败 / 已定稿 / 不适用 |
| 海报预览 | 附件/链接 | 海报产物图片 URL |
| 推文预览 | 多行文本 | 推文草稿正文 |
| 视频预览 | 附件/链接 | 视频产物 URL |
| 检查报告_海报 | 多行文本 | 海报审核报告 JSON 摘要 |
| 检查报告_推文 | 多行文本 | 推文审核报告 JSON 摘要 |
| 检查报告_视频 | 多行文本 | 视频审核报告 JSON 摘要 |
| run_id | 单行文本 | 全局唯一追踪 ID |
| content_version | 数字 | 生成版本号（再生成递增） |
| 触发时间 | 日期时间 | WF-0 触发时间戳 |
| 完成时间_海报 | 日期时间 | 海报生成的结束时间 |
| 完成时间_推文 | 日期时间 | 推文生成的结束时间 |
| 完成时间_视频 | 日期时间 | 视频生成的结束时间 |
| 审核决策_海报 | 单选 | 通过 / 修改 / 驳回 |
| 审核决策_推文 | 单选 | 通过 / 修改 / 驳回 |
| 审核决策_视频 | 单选 | 通过 / 修改 / 驳回 |
| 审核备注 | 多行文本 | 审核人的修改意见 |
